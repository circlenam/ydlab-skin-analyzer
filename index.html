About

YD Lab이 만드는
피부·두피 환경 분석의 새 기준

재능대학교 YD Lab은 환경오염과 피부·두피 건강의 상관관계를 정량화하고, AI 기반 맞춤 화장품 처방 시스템을 연구·개발합니다.

Technology

4가지 핵심 기술

단순 앱이 아닙니다. 연구 목적의 데이터 파이프라인과 AI 분석 엔진이 통합되어 있습니다.

🤖

LLM Vision AI 분석

Claude Sonnet(Anthropic) 기반 비전 AI가 스마트폰 현미경 이미지를 분석합니다. 피부 5가지 지표(주름·모공·결·톤·수분)와 두피 7가지 지표(각질·모공·모발굵기·두피색·수분밸런스·모발손상·탈모위험)를 각각 0~100점으로 정량화합니다.

🌫️

환경노출지수 — CEEI · SEEI

에어코리아 실측 API 데이터(PM2.5·PM10·NO₂·O₃)와 사용자의 거주기간을 결합해 두 가지 고유 지수를 산출합니다. 피부에는 CEEI(누적 PM2.5 × 거주기간), 두피에는 SEEI(복합 가중 지수 × 계절·UV·습도 보정)를 별도 적용합니다.

맞춤 화장품 성분 처방

AI 피부 스코어와 CEEI·SEEI 등급을 조합해 항산화·장벽강화·미백·진정 등의 기능성 성분을 자동으로 처방합니다. CEEI 150 이상 고노출 등급에는 항산화 성분이 우선 배치되며, 혼합 비율 가이드와 30 mL 세럼 제조 레시피까지 생성합니다.

How to Use

단 5단계로 완성되는
피부·두피 환경 분석

체험 이벤트 코드만 있으면 누구나 무료로 이용할 수 있습니다.

1

이벤트 코드 입력

체험 이벤트 참가 시 안내받은 코드를 입력합니다. 코드 없이는 접속이 제한됩니다.

2

모드 선택

분석 목적에 따라 모드를 선택합니다.

🌿 피부 분석 💆 두피 분석
3

기본 정보 · 생활습관 입력

성별·연령·거주 지역·거주기간 입력 후 자외선차단제 사용 여부, 흡연, 수면시간을 기록합니다.

4

사진 촬영 · 업로드

스마트폰 현미경 접안렌즈로 촬영한 사진을 업로드합니다. 부위별 1~3장 권장.

5

AI 분석 · 처방 확인

약 15~30초 후 피부·두피 스코어, CEEI·SEEI 등급, 맞춤 성분 처방, 세럼 레시피를 확인합니다.

FAQ

자주 묻는 질문

이벤트 코드는 어떻게 받나요? ▼
YD Lab 체험 이벤트 현장(재능대학교 관련 행사, 뷰티·화장품 박람회 등)에서 안내받을 수 있습니다. 온라인 문의는 아래 이메일로 연락주세요.
어떤 현미경을 사용해야 하나요? ▼
스마트폰 카메라에 부착하는 클립형 현미경 렌즈(50x~100x)를 사용합니다. 행사 현장에서 기기를 직접 제공하며, 직접 구매 시 약 1~3만원대 제품으로 이용 가능합니다.
CEEI와 SEEI는 무엇인가요? ▼
CEEI(Cutaneous Environmental Exposure Index)는 피부 전용 환경노출지수로, 거주 지역의 PM2.5 연평균값과 거주기간을 곱해 산출합니다. SEEI(Scalp Environmental Exposure Index)는 두피 전용 지수로, PM2.5·PM10·NO₂·O₃ 4종 오염물질 가중합에 계절·UV지수·습도 보정을 추가 적용합니다. 두 지수 모두 에어코리아 실측 API 데이터를 기반으로 합니다.
분석 결과는 의료 진단인가요? ▼
아닙니다. 본 시스템은 AI 기반 피부·두피 상태 분석 및 연구 목적의 데이터 수집 도구입니다. 의료적 진단이나 처방을 대체하지 않으며, 피부 이상이 의심되면 반드시 피부과 전문의를 방문하시기 바랍니다.
수집된 데이터는 어디에 사용되나요? ▼
IRB 승인 후 환경오염과 피부·두피 건강의 상관관계를 규명하는 SCI 논문 연구에 익명으로 활용됩니다. 개인을 특정할 수 없는 형태로 처리되며, 마케팅 동의 시에만 별도 연락이 발송됩니다.
피부 분석과 두피 분석을 동시에 받을 수 있나요? ▼
현재는 한 세션에서 피부 또는 두피 중 하나의 모드만 선택해 분석할 수 있습니다. 두 가지 모두 원하시면 코드를 재입력하거나 별도 세션으로 진행해 주세요.
